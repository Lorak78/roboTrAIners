## Introduction to TurtleBot
TurtleBot is a standardized robotic platform developed for ROS education and research. The concept of the TurtleBot platform is derived from Turtle robots used to teach foundational robotics and computer science since the early 1940s. TurtleBot is designed as a simplified, easily upgradable platform to teach people who are new to ROS, and to provide a capable base system for more advanced development. ## Languages and technologies used
TurtleBot is compatible with Python using **RosPy**, then to build programs you may use the commands listed in the guide (such as `catkin_make`)
